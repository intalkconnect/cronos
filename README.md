# Cronos (sem Tailwind)

Painel React (Vite + TS) para administrar os serviços do provisionador.

## Rodar em dev
```bash
npm i
npm run dev
# http://localhost:5173
```

## Build
```bash
npm run build
npm run preview
```

Na UI, configure a Base URL do provisioner (ex.: http://SEU_HOST:18080), informe o TENANT e use as ações.
